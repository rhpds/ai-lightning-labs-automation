## Storage

{{- if .Values.persistence.enabled }}
Persistent storage is enabled with PVC:
  Name: {{ include "openclaw.fullname" . }}-data
  Size: {{ .Values.persistence.size }}
  StorageClass: {{ .Values.persistence.storageClass | default "(default)" }}
{{- else }}
Persistent storage is DISABLED. Data will be lost when pods restart!
To enable persistence, set persistence.enabled=true in values.yaml.

## Building ClawSuite

ClawSuite will be built from source. To trigger the build:

  oc start-build {{ include "openclaw.fullname" . }}-clawsuite -n {{ .Release.Namespace }}

Or wait for the ImageChange trigger to fire automatically.

WARNING: The first build takes approximately 10 minutes to complete.

To watch build progress:

  # List all builds to find the build name:
  oc get builds -n {{ .Release.Namespace }}

  # Watch logs of the specific build:
  oc logs -f build/{{ include "openclaw.fullname" . }}-clawsuite-1 -n {{ .Release.Namespace }}
{{- end }}
{{- end }}

## Configuration

The OpenClaw configuration is stored in a ConfigMap:

  oc get configmap {{ include "openclaw.fullname" . }}-config -n {{ .Release.Namespace }} -o yaml

To update the configuration, edit values.yaml and run helm upgrade, or:

  oc edit configmap {{ include "openclaw.fullname" . }}-config -n {{ .Release.Namespace }}
